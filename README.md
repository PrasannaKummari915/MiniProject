# 🚀 FeatureFlow AI

> AI-Powered Feature Request & Voting System

FeatureFlow AI is a modern SaaS-inspired platform that allows users to submit feature requests, vote on ideas, collaborate in real-time, and track product updates through interactive roadmaps and changelogs.

Designed with a futuristic UI/UX, AI-powered insights, and real-time collaboration features, the platform improves communication between users and product teams.

---

# 🌟 Features

## 🔐 Authentication
- User Registration
- Secure Login
- JWT Authentication
- Protected Dashboard Routes

## 💡 Feature Request System
- Submit Feature Requests
- Upvote / Downvote Features
- Real-Time Comments
- Feature Status Tracking

## 🤖 AI Features
- Duplicate Feature Detection
